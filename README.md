# Менеджер паролей

## Описание проекта
Десктопное приложение для безопасного хранения и управления паролями. Все данные хранятся в зашифрованном виде с использованием мастер-пароля.

## Основной функционал
- Хранение логинов и паролей в зашифрованной базе
- Генератор надёжных паролей
- Автозаполнение на сайтах (опционально)
- Поиск и сортировка записей
- Категории для паролей (работа, личное, финансы)
- Экспорт/импорт данных (зашифрованный)

## Структура репозитория
- **src/main** - основной процесс Electron
- **src/renderer** - интерфейс приложения
- **src/shared** - общие модули и утилиты
- **src/encryption** - модули шифрования и дешифрования
- **docs** - документация
- **tests** - тесты
- **build** - сборки приложения
- **assets** - иконки, темы, ресурсы
